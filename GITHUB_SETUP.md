# 📤 SUBIR A GITHUB EN 5 MINUTOS

## **Paso 1: Crea una cuenta en GitHub (si no la tienes)**
1. Ve a https://github.com/signup
2. Crea tu cuenta con email
3. Verifica tu correo

---

## **Paso 2: Crea un nuevo repositorio**
1. Ve a https://github.com/new
2. Nombre del repo: `icecream-factura-pro`
3. Descripción: `Sistema de facturación para heladerías con Zebra ZQ320`
4. Elige **Public** (para que otros puedan verlo)
5. Presiona **Create repository**

---

## **Paso 3: Sube los archivos**

### **Opción A: Desde la web (MÁS FÁCIL)**

1. En tu nuevo repo en GitHub, presiona **Add file** → **Upload files**
2. Arrastra y suelta estos archivos:
   - `index.html` ✅
   - `manifest.json` ✅
   - `sw.js` ✅
   - `print-server.js` ✅
   - `package.json` ✅
   - `.gitignore` ✅
   - `README.md` ✅
   - `INSTALACION_RAPIDA.md` ✅

3. Presiona **Commit changes**
4. ¡Listo!

### **Opción B: Desde tu computadora (si tienes Git)**

1. Abre Terminal/PowerShell
2. Ve a la carpeta con tus archivos:
   ```bash
   cd /ruta/a/tus/archivos
   ```

3. Inicializa Git:
   ```bash
   git init
   git add .
   git commit -m "Initial commit: IceCream Factura Pro - Sistema de facturación"
   git branch -M main
   git remote add origin https://github.com/TU_USUARIO/icecream-factura-pro.git
   git push -u origin main
   ```

   *(Reemplaza `TU_USUARIO` con tu nombre de usuario en GitHub)*

---

## **Paso 4: Comparte tu repo**

Copia este link:
```
https://github.com/TU_USUARIO/icecream-factura-pro
```

Comparte con tus amigos/colegas para que descarguen.

---

## **Paso 5: Descarga desde GitHub en tu iPhone**

1. **Abre Safari en tu iPhone**
2. **Ve a:** `https://github.com/TU_USUARIO/icecream-factura-pro`
3. **Presiona el botón Code** (botón verde)
4. **Download ZIP**
5. Se descarga en tus **Archivos**
6. **Descomprime** la carpeta
7. **Busca `index.html`**
8. **Abre con Safari**
9. **Presiona compartir → Agregar a pantalla de inicio**

¡**LISTO! Ya tienes la app en tu pantalla de inicio del iPhone!**

---

## **Actualizar el repo después**

Si agregas nuevas características:

```bash
git add .
git commit -m "Descripción de cambios"
git push
```

---

## **🎯 URL FINAL**

